## 1.盒子模型：
### CSS的三大特性：
- 层叠性
- 继承性
- 优先性
### 2.边框(border)/内边距(padding)会影响盒子(div)实际大小
### 3.外边距可以让块级盒子水平居中，但必须满足：
- 盒子指定宽度
- 盒子左右外边距都设置为auto
### 4.行内元素或者行内块元素水平居中:
- 给其父元素添加 text-align:center
### 5.嵌套块元素塌陷问题解决方案：
- 可以为父元素定义上边框。
- 可以为父元素定义上内边距。
- 可以为父元素添加overflow:hidden。
### 6.清除内外边距：(重置浏览器默认样式)
- *{
    padding: 0;
    margin: 0;
}
